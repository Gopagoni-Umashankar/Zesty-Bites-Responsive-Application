# 🍽️ Zesty Bites

A responsive food ordering web application that allows users to register, log in, browse menu items, manage their cart, and place orders through an intuitive and user-friendly interface.

## 🌐 Live Demo

🔗 https://zesty-bites.netlify.app/

## 📌 Project Overview

Zesty Bites is a front-end restaurant ordering application developed using HTML, CSS, and JavaScript. The application provides a seamless food ordering experience by allowing users to explore menu items, manage their cart, and interact with a responsive user interface across different devices.

The primary goal of this project was to strengthen my understanding of front-end development concepts such as responsive web design, DOM manipulation, local storage, event handling, and form validation.

## ⚙️ Application Workflow

### Registration Process

1. User enters registration details.
2. Input validations are performed.
3. User data is stored in Local Storage.
4. User account is successfully created.

### Login Process

1. User enters login credentials.
2. Credentials are validated against stored user data.
3. On successful authentication, the user is redirected to the Home Page.

### Food Ordering Process

1. User browses available menu items.
2. User selects desired food items.
3. Items are added to the cart.
4. Quantity can be increased or decreased.
5. Cart data is automatically saved in Local Storage.

### Cart Management

- Add food items to cart
- Update quantities
- Remove items
- Maintain cart state after page refresh
